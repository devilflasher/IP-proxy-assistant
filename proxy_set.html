批量导入说明

名称位置建议填窗口编号，这样插件连接代理后，浏览器插件图标上会有角标显示窗口编号。

您可以通过批量导入功能一次性添加多个代理。请按照以下格式准备 CSV 文件：

名称 代理类型 IP地址 端口 用户名 密码
代理1 http 192.168.1.1 8080 user1 pass1
代理2 https 10.0.0.1 3128 user2 pass2

注意事项：

  • 支持 CSV 格式文件（可在 Excel 中另存为 CSV 格式）
  • 从第一行开始直接填写内容，无需表头
  • 按顺序填写：名称、协议、IP地址、端口、用户名、密码
  • 协议仅支持 HTTP、HTTPS 或 SOCKS5，不区分大小写
  • 端口号必须是 1-65535 之间的数字
  • 用户名和密码可以为空
  • SOCKS5 协议特别说明：目前仅支持无用户名密码的形式，这是因为 Chrome 扩展的代理 API 限制，不支持 SOCKS5 协议的认证方式
